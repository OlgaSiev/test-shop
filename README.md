# Product Catalog Test Task

React + Vite application for displaying products with:

- product cards
- sorting
- search filtering
- pagination
- volume selection
- add to cart UI

## Install

npm install

## Run project

npm run dev

## Build

npm run build

## Stack

React  
Vite  
SCSS  
Axios
